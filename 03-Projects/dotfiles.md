# dotfiles  [ПЛАН — репо ещё нет]

Точка входа в систему через интерфейс OpenCode. Управление конфигами Manjaro + аудит установленного софта.

**Окружение:** Manjaro (Arch-based). Менеджер дотфайлов — [проверить: GNU Stow / bare git / chezmoi?].
**CI / проверка:** нет (это конфиги, не приложение).

## Замысел
- OpenCode здесь = пульт управления системой, не разработка кода.
- Главный агент: **sysop** — ходит по всему $HOME и корню (read-only), инспектирует систему.
- Задачи: инвентаризация пакетов (pacman -Qe / -Qm), софт вне дотфайлов, дрейф конфигов.

## sysop (план, живёт в dotfiles/.opencode/agent/sysop.md)
- `external_directory: allow` (читает всю систему)
- `edit`: почти весь deny, пишет только отчёты
- bash whitelist: ls/cat/grep/pacman -Q*/which/systemctl status/stow -n
- НИКОГДА не ставит/удаляет пакеты и не правит /etc — только предлагает команды текстом

## Команды (план)
/sysaudit — софт вне дотфайлов + дрейф конфигов → отчёт
