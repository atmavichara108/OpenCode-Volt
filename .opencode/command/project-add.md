Команда: /project-add $ARGUMENTS

Создай карточку нового проекта в волте:

1. **Проверь** — нет ли уже карточки `03-Projects/$1.md`. Если есть — скажи и остановись.
2. **Создай** `03-Projects/$1.md` с frontmatter:
   ```yaml
   ---
   type: project
   repo: $2
   kind: $4 (по умолчанию — система)
   stack: $5 (по умолчанию — markdown)
   ---
   ```
   И кратким содержимым: заголовок, описание из `$3`, чеклист состояния.
3. **Добавь** проект в таблицу `00-INDEX.md` (секция ## Проекты) — вставь строку в алфавитном порядке.
4. **Запиши** в `04-Memory/facts.md`: проект `$1` добавлен, путь `$2`.
5. **Обнови** `active-context.md` — добавь проект в список отслеживаемых.
6. **Скажи** что сделано и что нужно донастроить (агенты, команды, методы).
