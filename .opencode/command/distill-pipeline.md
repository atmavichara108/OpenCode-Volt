# Дистилляция пайплайнов

Задача: $ARGUMENTS

## Шаг 1: Сбор данных

Для каждого проекта из `03-Projects/`:
1. Прочитай карточку проекта
2. Определи все команды (.opencode/command/)
3. Определи всех агентов (.opencode/agent/ и .opencode/subagent/)
4. Определи пайплайны (какие команды вызывают каких агентов)

## Шаг 2: Документирование

Оформи отчёт:

```markdown
# Дистилляция пайплайнов — YYYY-MM-DD

## Проект: <имя>
### Агенты
| Агент | Mode | Модель | Назначение |
|-------|------|--------|-----------|
...

### Команды
| Команда | Пайплайн | Назначение |
|---------|----------|-----------|
...

### Пайплайны
<схема: команда → агент → агент → ...>
```

## Шаг 3: Обновление волта

1. Обнови карточку проекта в `03-Projects/<name>.md`
2. Обнови таблицу методов в `00-INDEX.md`
3. Обнови `04-Memory/facts.md`
4. Если новый паттерн — создай метод в `02-Methods/`
5. Обнови `VibeOS.md`

## Шаг 4: Финализация

- `git add` всех изменённых файлов
- Предложи коммит-месседж: `feat(vault): дистилляция пайплайнов — <проекты>`
