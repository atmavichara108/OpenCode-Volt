Команда: /audit $ARGUMENTS

1. **Собери список проектов.** Если $1 указан — только `03-Projects/$1.md`. Если нет — все `.md` файлы из `03-Projects/`, кроме `vault.md`.

2. **Для каждого проекта:**
   - Прочитай карточку `03-Projects/<project>.md` → извлеки `repo:` путь.
   - Перейди в репозиторий (`ls <repo>` чтобы убедиться что существует).
   - Выполни `git pull --ff-only` (если есть remote).
   - Проверь `git log --oneline -3`.
   - Сверь с карточкой: какие агенты, команды, плагины есть в `.opencode/` репо? Устарело ли что-то в карточке?
   - Проверь статусы методов в карточке — не появилось ли чего нового в коде?

3. **Выведи сводную таблицу:**
   | Проект | Путь | Последний коммит | Статус карточки | Методы |
   |--------|------|-----------------|----------------|--------|

4. Если карточка устарела — предложи конкретные правки. Применяй только после подтверждения.
