# 05-Templates — Шаблоны для проектов и методов

Клонируемые заготовки для быстрого старта.

| Файл | Назначение |
|------|-----------|
| `project-card.md` | Шаблон карточки проекта в `03-Projects/` |
| `method.md` | Шаблон метода в `02-Methods/` |

## Использование

```bash
cp 05-Templates/project-card.md 03-Projects/НовыйПроект.md
# заполнить frontmatter и секции
```

Или через команду `/project-add` (создаёт карточку автоматически).
