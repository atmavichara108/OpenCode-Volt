# OpenCode: Плагины

> На основе реальных плагинов из моих проектов. Официальную спеку API — [проверить] на opencode.ai/docs.

## Что это
Lifecycle-хуки в `.opencode/plugins/`. Перехватывают события (перед bash, при компакции и т.д.).
В моих проектах используются:

| Плагин | Где | Назначение |
|--------|-----|-----------|
| env-guard | SERPlux (.js), dv-hub (.ts) | блокирует доступ агента к .env / секретам |
| notify | SERPlux (.js), dv-hub (.ts) | уведомления (звук/сообщение) о событиях |
| compaction | dv-hub (.ts) | управление сжатием длинного контекста |

## Заметка
- В SERPlux плагины на `.js`, в dv-hub на `.ts` — стоит унифицировать [проверить какой формат предпочтительнее].
- Содержимое самих плагинов в волт не копируем — живёт в репо. Здесь только реестр.
