## Ключевые решения
1. **ui-dev назван так** (не ux-dev) — чтобы соответствовать спецификации «developer», а не «designer»
2. **infra-dev на DeepSeek** — дешёвая модель для инфра-задач, не требует высокого качества
3. **Команды через .opencode/command/*.md** — стандартный подход OpenCode, не через opencode.json
4. **Агенты через .opencode/agents/*.md** — auto-discovery, не нужно дублировать в opencode.json
